# 翻刻ツールボックス

![ロゴ](./assets/honkoku.png)

日本の古典籍を翻刻する時によく利用する入力補助機能を集めたツールボックスです。

![パネル](./docs/panel.png)

## サポートされているサイト

- [みんなで翻刻](https://honkoku.org/app/)
- [日本語版Wikisource](https://ja.wikisource.org/)

## 使い方

### インストール

UserScript を入れてください。

### 開く方法

翻刻サイトのエディタを開くと右に「飜」のロゴ文字が表示されます。

![開く方法](./docs/open.gif)

自動的に表示されます。

## 開発

```
bun i
bun dev
```
